# ZERO Snap-On Cable Winder

A lightweight modular cable organizer designed for portable setups, maker desks, and evolving hardware systems.

This project started from two separate community suggestions:

* one proposal for a compact cable winder
* another idea exploring how the camera hole on ZERO could function as a mechanical attachment point

Both ideas were merged into a simple snap-on accessory optimized for:

* fast 3D printing
* low material usage
* clean cable management
* modular experimentation

The accessory attaches directly to ZERO but can also be used as a standalone cable organizer.

# Versions

## Small

Optimized for short and thin cables.

## Big

Deeper body designed for longer or bulkier cables.

Both versions can be resized depending on your setup and cable type.

# Print Settings

| Setting      | Value            |
| ------------ | ---------------- |
| Layer Height | 0.2 mm           |
| Material     | PLA / PETG / ABS |
| Infill       | 15–20%           |
| Supports     | Not Required     |

Recommended print orientation:

* flat on the base for optimal snap-fit behavior

# About ZERO

ZERO is a modular battery system designed to support community-created accessories and open hardware experimentation.

This project explores how small functional add-ons can evolve into a larger ecosystem of modular hardware components shaped alongside the community.
